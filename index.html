Por que usar o PIX Content Plugin?

100% Seguro

Sem requisições externas para gerar os QR Codes. O código é gerado offline no próprio navegador do cliente por segurança.

Sem Dependências Pesadas

Utiliza o moderno Web Asset Manager do Joomla 4/5/6 para carregar os scripts necessários sem poluir o cabeçalho.

Extremamente Rápido

Mapeamento direto por regex que converte os atalhos rapidamente durante o carregamento do artigo.

Guia Rápido de Uso

Sintaxe Básica

Adicione a tag abaixo em qualquer artigo ou módulo personalizado do Joomla (a moeda BRL é o padrão e tornou-se opcional):

{pix:VALOR|IDENTIFICADOR}

Ou explicitando a moeda:

{pix:MOEDA|VALOR|IDENTIFICADOR}

Parâmetros da Tag

Parâmetro Descrição Exemplo
MOEDA Opcional. A moeda do pagamento. Caso não seja informada, assume BRL como padrão. BRL
VALOR O valor do pagamento (use ponto para separar decimais). 890.00
IDENTIFICADOR Mensagem de identificação do pagamento (sem caracteres especiais). RelogioTitanium

Servidor de Atualização

Este plugin possui suporte nativo para atualizações automáticas diretamente no painel do Joomla através da url configurada no manifesto:
