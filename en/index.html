Why use the PIX Content Plugin?

100% Secure

No external API calls to generate the QR Codes. The generation happens completely offline in the customer's browser for security.

No Heavy Dependencies

Uses Joomla's modern Web Asset Manager to load necessary scripts cleanly without polluting the header.

Extremely Fast

Direct regex mapping that converts shortcodes instantly during article loading.

Quick Usage Guide

Basic Syntax

Add the tag below inside any Joomla article or custom module (BRL currency is the default and is optional):

{pix:AMOUNT|IDENTIFIER}

Or explicitly stating the currency:

{pix:CURRENCY|AMOUNT|IDENTIFIER}

Tag Parameters

Parameter Description Example
CURRENCY Optional. Payment currency. Defaults to BRL. BRL
AMOUNT The payment amount (use dot as decimal separator). 890.00
IDENTIFIER Payment identification message (no special characters). RelogioTitanium
